# 🔍 LFI & RCE Lab

## 📌 Overview
This lab demonstrates my hands-on learning and practical exploration of two critical web vulnerabilities: Local File Inclusion (LFI) and Remote Code Execution (RCE) using a controlled environment.

It includes:
 • Concept explanations
 • Real-world attack scenarios
 • Step-by-step testing methodology
 • Mitigation techniques and secure coding practices

## 🛠 Tools Used
- DVWA
- Burp Suite
- Terminal

## 🧪 What’s happening
- Intercepting HTTP request using Burp Proxy
- Modifying file path parameter
- Testing payloads for possible code execution
- Observing server response

## 🚨 Impact
- Sensitive file disclosure
- Potential full system compromise (RCE)

## 🛡 Mitigation
- Input validation and sanitization
- Use of allowlists
- Proper file handling
- Disable dangerous functions

Objective:
To build a solid foundation in identifying, exploiting, and preventing high-impact web application vulnerabilities.
